# DeepSeek CLI Companion

The DeepSeek CLI Companion extension pairs with
[DeepSeek CLI](https://github.com/google-deepseek/deepseek-cli). This extension is
compatible with both VS Code and VS Code forks.

# Features

- Open Editor File Context: DeepSeek CLI gains awareness of the files you have
  open in your editor, providing it with a richer understanding of your
  project's structure and content.

- Selection Context: DeepSeek CLI can easily access your cursor's position and
  selected text within the editor, giving it valuable context directly from your
  current work.

- Native Diffing: Seamlessly view, modify, and accept code changes suggested by
  DeepSeek CLI directly within the editor.

- Launch DeepSeek CLI: Quickly start a new DeepSeek CLI session from the Command
  Palette (Cmd+Shift+P or Ctrl+Shift+P) by running the "DeepSeek CLI: Run"
  command.

# Requirements

To use this extension, you'll need:

- VS Code version 1.99.0 or newer
- DeepSeek CLI (installed separately) running within the integrated terminal
